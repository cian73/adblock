# ----------------------
# Internal / DNS Rebinding Protection
# ----------------------

## IPv4 Private Networks
/^10\.(?:\d{1,3})\.(?:\d{1,3})\.(?:\d{1,3})$/
/^172\.(?:1[6-9]|2\d|3[0-1])\.(?:\d{1,3})\.(?:\d{1,3})$/
/^192\.168\.(?:\d{1,3})\.(?:\d{1,3})$/

## IPv4 Special / Link-Local / Loopback
/^169\.254\.(?:\d{1,3})\.(?:\d{1,3})$/
/^127\.(?:\d{1,3})\.(?:\d{1,3})\.(?:\d{1,3})$/
/^0\.0\.0\.(?:\d{1,3})$/

## IPv6 Private / Link-Local / Loopback
/^f[cd][0-9a-f]{2}:/          # ULA (fc00::/7)
/^fe80:/                      # Link-local
/^::1$/                       # Loopback
/^::$/                        # Unspecified

## Localhost / Local Domains
/^([a-z0-9\-]+\.)?(localhost|localdomain|ip6-localhost)$/
